###  Ensibuuko Test
Dockerize PHP Laravel API  Backend &amp; Frontend

#### Requirements
##### Install  the following
   - PHP 7.2
   - [docker](https://www.docker.com/) 
   - composer (php dependency manager)
   - vue-cli (frontend app)
   - Node 10.15

#### To run
- Docker bulild
   ```
   $ docker-compose build
   ```
- Spin up the containers
  ```
  $ docker-compose up -d
  ```
